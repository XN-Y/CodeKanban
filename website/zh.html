多终端统一管理

一个界面管理所有项目的终端。支持多标签页、拖拽排序，快捷键 ` 秒开终端面板。

可视化任务看板

Kanban 风格任务管理，支持拖拽移动任务状态。任务可关联终端和 Git 分支，一目了然。

AI 会话历史

自动记录与 AI 工具的所有对话。一键回溯代码修改历程，快速定位任何一次 AI 交互。

Git Worktree 管理

可视化管理 Git Worktree 分支。支持创建、合并、删除分支，让多任务并行开发更轻松。

多标签笔记

快捷键 1 即开笔记。支持项目级/全局笔记切换，粘贴图片，自动保存。
